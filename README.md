# Polynomial Regression

A beginner-friendly implementation of **Polynomial Regression** using **scikit-learn** on a synthetic dataset.

## Features

- Generate a dataset (of any degree, but make sure, you update that using the polynomial features. I chose a quadratic dataset using np.random)
- Create polynomial features
- Train a Linear Regression model
- Evaluate using the R² score
- Visualize the fitted polynomial curve

## Tech Stack

- Python
- NumPy
- Matplotlib
- scikit-learn
